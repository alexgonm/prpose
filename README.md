# Forum de Discussions Structurées

Projet du cours Projet Transversal 1 (Printemps 2019)

## Présentation

Ce projet a été effectué dans le cadre du séminaire Projet Transversal 1 et consiste à créer un site similaire à Reddit et Inilab, une plateforme de vote pouvant prendre en compte des publications et des commentaires. Le site est également supposé être capable de donner une catégorie à chaque publication automatiquement.

Le site devrait également être capable d'afficher les publications et les commentaires de manières triées selon les votes qu'ils ont reçu.

## Structure du projet

Le projet est découpé en deux parties. L'une, le [client](https://github.com/granpepito/prpose/tree/master/client) est l'interface permettant aux utilisateurs d'utiliser la plateforme. L'autre, le [serveur](https://github.com/granpepito/prpose/tree/master/server), est la partie qui s'occupera des traitements de données et de la communication à la base de données.
